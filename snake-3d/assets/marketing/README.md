# Hellcoil Android Market Assets

These images are sized and formatted for Google Play Store upload.
Place in the listing using the slot names below.

## Store Assets

| File | Slot | Size | Notes |
|---|---|---|---|
| `hellcoil_app_icon_512x512.png` | App icon | 512x512 | RGBA, circle-safe |
| `hellcoil_feature_graphic_1024x500.png` | Feature graphic | 1024x500 | Top of store page |
| `hellcoil_screenshot_1_gameplay_1080x1920.png` | Screenshot 1 | 1080x1920 | Gameplay |
| `hellcoil_screenshot_2_action_1080x1920.png` | Screenshot 2 | 1080x1920 | Action / bullet hell |
| `hellcoil_screenshot_3_title_1080x1920.png` | Screenshot 3 | 1080x1920 | Title screen + leaderboards |

## Style
- Retro 80s/90s arcade neon
- Palette: hot pink, electric cyan, lime, orange on near-black
- No small text (avoids misspellings)
- CRT glow / scanline aesthetic

## Upload checklist
1. App icon -> "App icon" field (round mask preview)
2. Feature graphic -> "Feature graphic" field
3. Screenshots 1-3 -> Phone screenshots (must be 1080x1920 portrait PNG)
4. Add at least 2 screenshots (Google requires 2 minimum; we have 3)
