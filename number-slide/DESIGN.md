# Number Slide — Sliding Tile Puzzle

## Concept
The classic 15-puzzle mechanic (public domain — no single product owns
generic sliding-tile play): slide numbered tiles into the one empty slot
to put them back in order, 1 through 15. A pure planning puzzle with no
timer and no reflexes, using the same shared-move-budget fail state as
Color Sort: solving refills moves, and each new puzzle is scrambled a
little harder, so efficient solving is what keeps a run alive.

## Aesthetic
- Flat 2D, dark background, 4x4 grid of blue numbered tiles with one
  dark empty slot.

## Mechanics (LOCKED - do NOT add more)
1. A 4x4 board (15 numbered tiles + 1 empty slot) is generated by making
   random legal slides from the solved state, guaranteeing it's always
   solvable.
2. Tap a tile orthogonally adjacent to the empty slot to slide it in.
   Each slide costs one move from a shared budget.
3. Solving the board (tiles back in order 1-15, empty last) refills the
   move budget and generates a new, more scrambled puzzle.
4. Running out of moves before solving ends the run. Score = puzzles
   solved. High score via config.
5. Novel element: Locked Tile. A rare numbered tile is fixed in place for
   the puzzle's first few moves (counted across the whole puzzle, not
   just moves that touch it) — it can't be the target of a slide even
   when adjacent to the empty slot, forcing a route around wherever it
   currently sits, tinted amber until it unlocks.
